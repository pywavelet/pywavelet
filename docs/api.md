# API

The main functionality of `pywavelet` is to help transform/inverse transform
data from the time/frequency domains to the WDM-wavelet domain.

## Transform Functions

- `pywavelet.transforms.from_time_to_wavelet`
- `pywavelet.transforms.from_freq_to_wavelet`
- `pywavelet.transforms.from_wavelet_to_freq`
- `pywavelet.transforms.from_wavelet_to_time`

## Types

- `pywavelet.types.TimeSeries`
- `pywavelet.types.FrequencySeries`
- `pywavelet.types.Wavelet`
