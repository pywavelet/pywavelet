Citing PyWavelet

Citing PyWavelet#

If you use PyWavelet in academic work, please cite the package and the underlying WDM-wavelet references.

BibTeX entries are maintained in docs/references.bib:

@misc{pywavelet,
    author = {Vajpeyi, Avi and Baghi, Quentin and Burke, Ollie and Mentasti, Giorgio and Speri, Lorenzo},
    title = {PyWavelet},
    url = {https://github.com/pywavelet/pywavelet/},
    year = {2023},
    note = {Version 0.2.10},
}

@ARTICLE{Cornish:2020:PhRvD,
       author = {{Cornish}, Neil J.},
       title = "{Time-frequency analysis of gravitational wave freqseries}",
       journal = {{Physical Review D}},
       keywords = {General Relativity and Quantum Cosmology, Astrophysics - High Energy Astrophysical Phenomena},
       year = 2020,
       month = dec,
       volume = {102},
       number = {12},
       eid = {124038},
       pages = {124038},
       doi = {10.1103/PhysRevD.102.124038},
       archivePrefix = {arXiv},
       eprint = {2009.00043},
       primaryClass = {gr-qc},
       adsurl = {https://ui.adsabs.harvard.edu/abs/2020PhRvD.102l4038C},
       adsnote = {Provided by the SAO/NASA Astrophysics Coupled Data System},
       url = {https://arxiv.org/abs/2009.00043},
}
